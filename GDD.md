3 Pilares de Design
Os pilares guiam toda decisão de design. Toda feature nova tem que servir a pelo menos um deles. Se não serve, é corte.

🏰 1. RECONSTRUÇÃO (o epicentro)
"Nada no jogo é isolado. Tudo empurra a vila pra virar reino."

Plantar uma cenoura → alimenta NPC → NPC desbloqueia quest → quest abre dungeon → dungeon dropa material → material constrói prédio → prédio atrai novo NPC.
A vila é o HUD emocional do progresso. O jogador vê o reino nascer na tela, não num menu.
Progressão de título narrativo: Errante → Fazendeiro → Artesão → Líder → Rei — cada degrau tem entrega visível no mundo.
Implicação de design: rejeitamos sistemas paralelos desconectados. Farming que não alimenta combate é corte. Crafting que não constrói vila é corte.
🧩 2. LIBERDADE MODULAR
"Tem caminho pra todo jogador, mas a economia amarra todos."

Weapon-as-class: a arma equipada define o playstyle (sem travas de classe).
Crafting por componentes: builds são montadas, não sorteadas.
Playstyles válidos: farmer puro que vende material, dungeon crawler que compra comida, trader que nunca pega numa espada — todos progridem.
Mapa procedural + dungeons variadas dão rejogabilidade sem roteiro forçado.
Implicação de design: economia player-driven é sagrada. Um jogador precisa poder viver do que o outro ignora.
⚔️ 3. RISCO COM PESO
"Toda ida à dungeon é uma decisão consciente. A morte dói, mas não destrói."

Mochila de Expedição limita o que você leva e o que traz — escolher é sofrer.
Estrela de Sangue dá peso permanente à morte sem soft-lock de progresso.
Build > reflexo: combate tático, não twitch. Preparação vence execução.
Dungeons são o gate de progressão mid/endgame — travadas por volume diário, então cada entrada importa.
Implicação de design: combate nunca é "mais uma run". Fora da dungeon, o tempo é do jogador. Dentro dela, o tempo tem preço.

Posicionamento Estratégico
[TÍTULO] ocupa o espaço entre Stardew Valley e Path of Exile 2 — não como compromisso entre os dois, mas como jogo onde o lar torna o combate significativo e o combate torna o lar possível. Para o jogador de 25–40 anos que quer profundidade tática sem sacrificar o senso de pertencimento, o projeto entrega um loop único: cada cenoura plantada empurra um reino, cada dungeon tem contexto emocional real. Enquanto No Rest for the Wicked captura o público hardcore isométrico e Palia captura o casual-online, este projeto ocupa a faixa subatendida do "life-sim player que cresceu e quer consequência" — um posicionamento ainda sem dono claro no mercado PC.

Risco Principal e Mitigação
Risco: Escopo excessivo para a fase de protótipo.

O design é construído sobre interdependência total ("tudo alimenta tudo"), o que é o maior diferencial — e o maior risco de execução. Se o sistema de economy player-driven ou o co-op atrasar, o loop central pode nunca ser validado.

Mitigação recomendada: definir um Vertical Slice como primeiro marco obrigatório — um único ciclo completo fazenda → NPC → dungeon → vila, numa área pequena, solo, sem co-op, sem endgame, sem economy global. O objetivo do Vertical Slice é validar o feeling do ciclo, não a escala. Tudo mais vem depois.

Etapa 3 — Core Loop & MDA/Tetrad
Core Loop, MDA & Elemental Tetrad — [TÍTULO PROVISÓRIO]
Verbo Principal & Fantasia de Verbo
Verbo-núcleo: RECONSTRUIR

A maioria dos ARPGs tem como verbo "matar"; os life-sims têm "cultivar". Aqui, o verbo que amarra tudo e que é exclusivo deste jogo é reconstruir — porque toda ação do jogador (plantar, esmagar, craftar, entregar) é um tijolo do reino que nasce na tela.

Verbos-satélite (por zona):

Vila/Fazenda: plantar, colher, forjar, entregar
Dungeon: esmagar, explodir, catar, extrair
Fantasia de verbo: "Eu reconstruo meu reino uma cenoura, uma forja e um cadáver de cada vez."

Core Loop em 3 Camadas
🔁 Micro-Loop (5–30s) — "o dedo no mouse"
O jogo tem dois micro-loops paralelos, um por zona. Cada um precisa ser satisfatório por si só, porque o jogador passa 80% do tempo dentro de um deles.

Micro-loop da Fazenda/Vila (Stardew-style):

click tile → ação (regar/plantar/cortar/minerar) → feedback (anim + som + partícula)
    → recurso cai no inventário → barra de skill/vila enche
    → hover próxima tile → click → ...
Micro-loop da Dungeon (PoE2-style):

detecta pack → skill/ataque → dodge → kill (explosão + loot)
    → pickup (brilho no chão) → verifica HP/Mana → próximo pack → ...
🔁 Loop Médio (30–300s) — "o encontro"
Na Fazenda: um mini-dia produtivo — regar fileira inteira → colher o que amadureceu → processar na forja/cozinha → estocar → preparar Mochila de Expedição pra próxima dungeon.

Na Dungeon: um andar/bioma — limpar todos os packs do layout → enfrentar elite/mini-boss → abrir chest especial → decisão crítica: descer mais fundo (mais loot, mais risco) ou dar extract agora com a mochila cheia?

🔁 Macro-Loop (300s+) — "o ciclo do reino"
Este é o ciclo fechado canônico do projeto, traduzido em gesto:

PLANTAR colheita específica
    ↓
ALIMENTAR / ENTREGAR a NPC (ou vender na economia global)
    ↓
NPC desbloqueia QUEST
    ↓
QUEST abre DUNGEON nova (ou modifica layout de uma existente)
    ↓
DUNGEON dropa MATERIAL raro
    ↓
MATERIAL vira INPUT de PRÉDIO (entregue automaticamente)
    ↓
PRÉDIO nasce na vila + atrai novo NPC + nova colheita/recurso desbloqueado
    ↓
(título narrativo sobe: Errante → Fazendeiro → Artesão → Líder → Rei)
    ↓
loop recomeça em escala maior
Tempo estimado de um macro-loop completo no early-game: 45min a 2h reais. No endgame: pode passar de 10h (Atlas-style).

Recursos / Economia Interna
O jogador gerencia 4 medidores em dungeon e 2 macro-recursos fora dela.

Dentro da Dungeon
Recurso	O que é	Pressão que cria
Vida	HP clássico; zero = morte	Execução, posicionamento
Mana	Custo de skills; regenera lento	Rotação de skills, não spammar
Espaço da Mochila de Expedição	Peso/slots limitados; escolher o que levar	"Pego este item raro e largo aquele?"
Estrela de Sangue	3 slots de morte por dungeon	Cada morte escala a dungeon e sobe o custo
Fora da Dungeon
Recurso	O que é	Pressão que cria
Materiais (colheita + loot + craft)	Input de tudo — quests, prédios, builds	Decidir o que plantar, o que craftar, o que vender
Moeda da economia player-driven	Ouro/créditos para comprar dos outros jogadores	"Será que é melhor farmar ou comprar?"

Os 2 Twists que Mantêm o Loop Fresco
Twist 1 — Estrela de Sangue como escalador adaptativo
Morrer não é "reset" nem "perde tudo imediato". É uma escada de aposta:

1ª morte: dungeon sobe 1 level, tua Mochila vira "cadáver-loot" no chão do ponto da morte, você volta ao hub.
2ª morte: dungeon sobe +1 level (total +2), o cadáver antigo continua lá, novo cadáver aparece.
3ª morte: game over da run — perde toda a mochila, perde direito a voltar, lockout de 12h real naquela dungeon específica.
Se você extract vivo, tem 24h reais pra voltar em qualquer cadáver anterior e resgatar.
Por que funciona mecanicamente: transforma cada morte num investimento-risco real, sem ser Souls-punitivo. E o lockout de 12h vira pressão narrativa, não pay-to-wait, porque a dungeon não é a única fonte de progresso (a vila está sempre aberta).

Twist 2 — Construção Assimétrica: Fazenda Livre, Vila Automática
Fazenda = expressão do jogador (Stardew): ele decide onde regar, onde plantar, onde pôr a forja. Loop médio de expressão.
Vila = espelho emocional (automática guiada): ele entrega recursos via quest, e a vila se constrói sozinha — o moinho aparece, o NPC chega, o letreiro da praça surge. O jogador vê o reino nascer sem micro-gerenciar.
Por que funciona: separa a expressão (fazenda) da narrativa (vila). Dá liberdade onde importa pro dia-a-dia e entrega recompensa visual cinemática onde importa pro coração.

Elemental Tetrad (Jesse Schell)
🔧 Mecânica
Click-to-attack ARPG + skills com cooldown + dodge + Mochila de Expedição com peso + Estrela de Sangue + farming tile-click + construção livre da fazenda + reconstrução automática da vila + weapon-as-class + crafting por componentes + economia player-driven. O núcleo de regras é o ciclo fechado: cada sistema é input de outro sistema.

📖 Narrativa
Herdeiro da Casa Varen, caído após a Fragmentação, recomeça num canteiro. A cada cultura colhida, a cada NPC que chega, a cada dungeon limpa, a vila renasce — e o caminho até os responsáveis pela queda se abre. A vingança não é declarada, é construída. O título narrativo sobe com o mundo visível: Errante → Fazendeiro → Artesão → Líder → Rei.

🎨 Estética (visual/som)
Dual-tone controlado:

Vila/Fazenda: pixel art quente, dourada, luz difusa, paleta tipo Witchbrook/Stardew. Som de folhas, sinos, madeira, passos em grama.
Dungeon: pixel art fria, neon + ruínas, contraste alto, paleta tipo Hyper Light Drifter/Nier. Som de synth distorcido, eco, metal contra pedra.
A transição entre as duas é diegética (você atravessa um portal/runa) — nunca via menu.
💻 Tecnologia
PC (Steam) · engine 2D isométrica · vila e NPCs artesanais (handcrafted) · dungeons e mapas de exploração procedurais · infra online pra co-op até 6 e economia player-driven global · anti-cheat leve (consequência: F2P exige isso). Vibe-coding é viável pro protótipo, mas os 4 elementos do Tetrad precisam bater antes do Vertical Slice.

Coerência do Tetrad: os 4 elementos se reforçam — a mecânica da reconstrução só faz sentido com a narrativa do luto; a estética dual-tone só funciona porque a tecnologia permite carregamento entre zonas curtas; a narrativa de "vila como HUD" só entrega com a estética quente visível.

Risco de Design Principal
A Mochila de Expedição + Estrela de Sangue podem se tornar fricção compulsiva em vez de tensão saborosa.

O desenho é elegante no papel — mas empilha três penalidades simultâneas em quem morre: (1) dungeon sobe de level, (2) cadáver-loot exige nova run pra resgatar, (3) na 3ª morte vem lockout de 12h. Se o balance não for cirúrgico, o jogador-alvo (25–40 anos, 2h por sessão) pode sentir que "morrer é castigado demais" e simplesmente parar de ir em dungeon, quebrando o macro-loop (vila não progride sem material de dungeon).

Mitigações a testar no protótipo:

A 1ª morte deve ser quase grátis — escalada de dungeon mínima, cadáver-loot próximo ao portal, fácil de resgatar. A punição cresce de forma não-linear, com a 3ª sendo o pico.
O lockout de 12h precisa ter saída narrativa (ex.: NPC da vila dá quest de "luto" que compensa parcialmente) pra não virar pay-to-wait mascarado.
A vila NUNCA trava por morte de dungeon. Farmer puro continua progredindo. Isso respeita o pilar "Liberdade Modular".
Risco secundário: o micro-loop da fazenda tem que ser satisfatório sozinho (sem depender da dungeon pra ser fun). Se plantar/colher for chore, o macro-loop desaba pelo elo mais fraco.

Ciclo: 4h real = dia / 4h real = noite (ajustável em balance).
Horário mostrado em UI (relógio estilizado no HUD).
2.2 Consequências Mecânicas
Período	Fazenda/Vila	Mapa de Exploração	Dungeons
Dia	NPCs trabalham; lojas abertas; quests diurnas	Mobs tier normal; drops normais	Sem mudança

2.3 Descanso (substitui "Dormir Regenerativo")
Tua proposta inicial foi "dormir recupera HP/MP + processa XP". Refatorado pra resolver as 3 armadilhas (ver abertura do turno):

Descanso é uma ação de 10s na cama da fazenda ou estalagem.
Não passa tempo do mundo — tempo MMO é imutável.
Aplica o buff "Descansado" por 30min reais: +10% XP ganha e +10% drop rate.
Regen de HP/MP é passiva e lenta fora de combate em qualquer lugar; descansar apenas acelera e dá o buff.
Cooldown do descanso: 1 vez por ciclo de 24h in-game (8h real).
Por que funciona: vira ritual de início de sessão ("ok, descansei, bora dungeon com buff"), não substitui a noite produtiva, não é idle-mechanics.

3. Reputação — A Mecânica Mestre de Progressão
Esta é a mecânica nova mais importante desta etapa. Ela amarra Reconstrução (Pilar 1) em números.

3.1 O que é
Reputação é um número único por jogador (0 → infinito, mas com patamares significativos). Não é faction; é prestígio dentro do reino.

3.2 Como ganha (3 fontes, balanceadas pra não favorecer nenhum playstyle)
Fonte	Como gera
Combate/Dungeon	Dungeons concluídas, bosses derrotados, tier da dungeon
Fazenda/Vila	m² cultivados, diversidade de culturas, edifícios erguidos
Gear Score (crafting)	Ítens de alto tier craftados (não equipados — craftados, o ferreiro puro ganha)
Regra de balanceamento inicial (pra ajustar): cada fonte contribui ~33% do total possível numa hora de jogo. Isso honra Liberdade Modular (Pilar 2) — farmer puro, crawler puro, crafter puro progridem em Reputação no mesmo passo.

3.3 O que a Reputação destrava
Patamar	Destrava
0–100 (Errante)	Fase 1, fazenda inicial, primeira dungeon
100–500 (Fazendeiro)	2º NPC da vila, 2º prédio, mapa de exploração adjacente
500–2000 (Artesão)	Crafting tier 2, novas armas, Atlas inicial de dungeons, AH global
2000–8000 (Líder)	Market player-driven, prédios especiais da vila, co-op party
8000+ (Rei)	Modo Reino completo, prédios raros, endgame Atlas-style, eventos de servidor
Os nomes de patamar espelham os títulos narrativos da Etapa 1. Reputação é o pulso numérico do título.

3.4 Reputação não sobe sozinha
Não há "daily login = +rep". Reputação só vem de ação significativa. Isto é um anti-F2P-predador: não ofertamos xp pass acelerado pago.

4. Combate — Especificação
4.1 Filosofia
Ritmo PoE2: rápido, combo-based, leitura de pack. Não é Souls lento nem Diablo-spam.

4.2 Specs Numéricos (primeiro draft, pra ajustar)
Parâmetro	Valor alvo
Dodge-roll	400ms duração, 200ms i-frames no meio, 600ms cooldown
Hit-stop em crit	80ms
Skill-use animation (média)	350–600ms
TTK em mob normal tier 1	1–3 hits
TTK em elite tier 1	8–15 hits
Boss tier 1	60–120s
4.3 Recursos em combate (confirmado: só Mana, sem estamina)
HP: regenera fora de combate; em combate só via poção/skill.
Mana: regenera lento e constante (inclusive em combate). Skills custam mana; auto-attack não.
Sem estamina → dodge é limitado por cooldown, não por recurso contínuo. Mais legível, menos micro-gestão.
4.4 5 Skills Ativas + 1 Auto-attack
Slot 0: auto-attack (vinculado à arma, sem custo)
Slots 1–5: skills equipáveis (ver §5)

5. Weapon-as-Class — Specs & Compatibilidade
5.1 Modelo "skills aprendidas via arma, equipáveis em armas compatíveis"

2 slots de arma em uso (troca rápida com tecla dedicada, sem pausa).
Skills equipadas são por loadout de arma (trocar arma troca os 5 skill-slots).

9. Economia Player-Driven
9.1 Moeda Única: Ouro
Só Ouro como moeda. Sem gold/credit/gem split.
Ouro é gerado via: vender para vendor NPC, quests, loot de dungeon, fee de mercado (ver §9.3).
Ouro não é comprável com dinheiro real. Nunca.
9.2 Trade sempre = Ouro + Item
Todo trade player-to-player é Ouro + item, nunca item-por-item puro. Isto:

Impede barter game-the-market-maker.
Dá ao Ouro demanda constante.
Simplifica AH UI.
9.3 Auction House Global (PoE-style)
Listagem: jogador anuncia item por valor em Ouro.
Fee: 5% do valor listado é queimado na transação (gold-sink, anti-inflação).
Busca: filtros por material, stat, Gear Score, tier.
Entrega: instantânea via sistema de correio.
9.4 Mercado Local da Vila (diferencial)
Cada jogador pode montar um kiosk na própria vila.
Outros jogadores podem visitar sua vila (modo visita) e comprar do kiosk.
Kiosk não tem fee de 5% (trade-off: só quem vai até lá compra).
Cria incentivo social real de visitar vilas dos outros → Fellowship (estética MDA) em forma de comércio.
9.5 Anti-inflação
Fee de 5% no AH (gold-sink principal).
Impostos de construção (prédios caros custam ouro + materiais).
Taxa de entrega no correio.
10. Dungeons — Especificação Detalhada
10.1 Estrutura de Módulos
Layout = grafo de 8–15 rooms em dungeon early.
Cada room é handcrafted (pré-desenhada pixel art).
O grafo é procedural: conexões entre rooms, seed de inimigos, loot e tile-art variations.
Variação estética: 4 biomas iniciais (Cripta, Floresta Corrompida, Mina Rúnica, Ruína Tech).
10.2 Duração Alvo
Early game: 15–20min por run completa.
Mid game: 30–45min.
Endgame (Atlas-style): 45min–2h com modificadores.
10.3 Estrutura Interna de uma Run
Entrada → 3-5 rooms de combate normal → 1 room-elite (mini-boss)
    → 2-4 rooms de puzzle/loot → sala do boss final
        → extract (portal de volta à vila)
10.4 Decisão Extract
Após o mini-boss, portal de extract opcional aparece. Jogador decide:

Extract agora: garantia de loot até aqui, sem boss-loot.
Continuar: pode pegar boss-loot mas arrisca (ver §11.2 regra do boss).
10.5 Boss Final
Sempre obrigatório pra concluir a dungeon.
Extract antes do boss = dungeon conta como "abandonada" (loot preservado, mas sem progresso de Atlas/Reputação de dungeon concluída).
11. Estrela de Sangue v2 — Reconciliada
11.1 Duas regras de morte (importante, reconciliação com Etapa 3)
Regra A — Morte por Mob/Trap (canonizada na Etapa 3):

1ª morte: dungeon +1 tier; cadáver-loot no chão; volta ao hub.
2ª morte: dungeon +1 tier adicional (+2 total); cadáver antigo + novo.
3ª morte: game over da run, perde a mochila toda, lockout 12h wall-clock real naquela dungeon específica.
Extract vivo = 24h wall-clock real pra resgatar cadáveres.
Regra B — Morte no Boss Final (nova, resolve P21):

Chances no boss = nº de Estrelas de Sangue não-usadas ao chegar ali.
Chegou com 3 estrelas: 3 chances.
Chegou com 1 estrela (morreu 2x pra mobs): 1 chance no boss.
Se morrer com todas as chances do boss gastas: Expulsão Honrada
Mantém a mochila (loot coletado até ali).
Dungeon fica marcada "não-concluída".
Não ganha boss-loot exclusivo nem Reputação de conclusão.
Pode retornar à dungeon depois (não há lockout de 12h nesse caso).
Se vencer o boss: dungeon "concluída", boss-loot exclusivo cai, Reputação maior.
11.2 Visibilidade do Lootbag
Lootbag / cadáver-loot é privado: só o dono vê e pode saquear.
Em co-op, cada player tem seu próprio cadáver e sua própria Estrela de Sangue individual.

15. Risco de Design Principal desta Etapa
Risco: Reputação como gate universal pode virar funil estreito.

Se toda progressão depende de Reputação e as 3 fontes não ficam perfeitamente balanceadas, um playstyle fica atrás. Se farming dá 2x mais Rep/hora que dungeon, ninguém faz dungeon. Vira "meta-farm".

Mitigação:

Telemetria obrigatória no Vertical Slice: medir Rep/hora por playstyle em 3 perfis-teste (farmer puro, crawler puro, crafter puro). Alvo: diferença <15% entre os três.
Soft caps por fonte: se o jogador ganhou 500 Rep só de farming hoje, a fonte farming rende 50% menos até ele fazer outra coisa. Incentiva diversidade sem forçar.
Quests ancoradas por fonte: quests específicas só dão Rep se concluídas via fonte certa (ex: "dungeon quest" só dá Rep se concluir dungeon, não dá se comprar o item no AH).
Risco secundário: MMO-time estático + ciclo dia/noite com mobs diferentes significa que um jogador de 22h só joga de noite no jogo. Precisa garantir que conteúdo noturno e diurno são equivalentes em quantidade e Rep gerada, não apenas diferentes.

ETAPA 5 — WORLDBUILDING
1. Tensão Central do Mundo
Eldara está sendo silenciosamente devorada por dentro pela cobiça de quem descobriu o que jamais deveria ter sido tocado.

A "Fragmentação" não foi um cataclismo cósmico — foi uma descoberta. Uma civilização antiga e tecnomágica habitou Eldara muito antes da era atual e deixou enterrado um saber capaz de fundir Cyber, Magia e Orgânico em uma só corrente de poder. Quando essa herança foi redescoberta pelas Grandes Casas há gerações, a corrida por seu controle reescreveu a política do continente.

A ferida do presente é o Golpe contra a Casa Varen, há 6 meses. Uma casa de segundo escalão, em ascensão veloz, foi exterminada num único golpe coordenado — porque ameaçava a oligarquia e porque guardava, sem saber, a chave biológica do próprio poder antigo: os Filhos de Varen, imunes à corrupção da fusão.

A ferida não cicatrizou. Está sangrando. E o sangue tem nome.

2. As 3 Eras
🜂 Era Antiga — A Civilização Antes do Nome
Muito antes da história registrada, uma civilização tecnomágica descobriu Eldara e a habitou. Eles dominaram a tríade — fundiram máquina, magia e carne sem se perderem. Desapareceram (motivo ainda submerso no iceberg: autoextinção pela fusão excessiva). Deixaram para trás ruínas com memória digital — as dungeons. Cada dungeon é um fragmento da consciência coletiva dessa civilização morta, ainda processando, ainda sonhando.

Implicação ludonarrativa: isso justifica por que dungeons têm "memória", drops cibernético-mágicos, e por que cada bioma de dungeon tem uma "voz" própria. Não é random — é o ECO de quem morava ali.

🜃 Era Recente — A Era das Sete Casas
Eldara foi colonizada por humanos que encontraram as ruínas. Da extração do metal raro (Veridium) e da fonte de energia primária (Lume Antigo) nasceu a aristocracia atual:

7 Grandes Casas controlam minas, rotas e o conhecimento da fusão tríade.
~50 Casas Menores orbitam, prestam vassalagem, ou tentam ascender.
Acima de tudo: o Concílio das Sete, uma oligarquia que finge ser conselho.
Foi nesta era que 3 das 7 Grandes Casas + ~20 Menores formaram a Conspiração do Crisol: derrubar qualquer Casa que ameaçasse o monopólio do poder antigo.

🜄 Era Presente — O Pulso Cortado
Há 6 meses, a Casa Varen caiu em uma única noite. Conhecida por lealdade e propósito (e por crescer rápido entre as Menores), Varen foi exterminada com cirurgia política. Os decendentes da casa Varen, representados por seus muitos filhos pelos quatro cantos de Eldara antes do golpe — em vilas pobres, beira de estrada, lugares onde ninguém olharia. A regente da casa foi capturada viva e está, hoje, sob posse do Líder Supremo do Outro Continente.

Plot crítico (revelação tardia): o Concílio acredita que matou todos os Filhos. O Líder Supremo sabe que escondeu vários — e está procurando.

3. Cosmologia — A Tríade
O poder de Eldara não é "magia" nem "tecnologia": é a Tríade.

Vetor	O que é	Quem manipula
Cyber	Runas-circuito, próteses senscientes, memória digital nas ruínas	Engenheiros do Concílio, Líder Supremo
Magia	Energia do Lume Antigo, canalizada por cajados rúnicos e rituais	Ordens arcanas das Casas
Orgânico	Carne, semente, sangue — vida cultivada e moldada	Fazendeiros, alquimistas, Casa Varen historicamente
A Lei Não-Escrita: misturar dois vetores corrompe o usuário. Misturar os três destrói o usuário — exceto se ele for um Filho de Varen. Eles são imunes pela linhagem. É por isso que a oligarquia precisava extinguir a Casa: enquanto Varen existir, alguém pode usar o poder antigo sem custo.

Conexão com mecânicas: o Crafting por Componentes (Etapa 4) reflete a Tríade — todo item de tier alto pede componente Cyber + Mágico + Orgânico. O protagonista craftando esses itens é, literalmente, a coisa que ele não pode ainda saber que é.

ETAPA 5 — WORLDBUILDING
1. Tensão Central do Mundo
Eldara está sendo silenciosamente devorada por dentro pela cobiça de quem descobriu o que jamais deveria ter sido tocado.

A "Fragmentação" não foi cataclismo — foi descoberta. Uma civilização antiga e tecnomágica habitou Eldara e deixou enterrado um saber capaz de fundir Cyber, Magia e Orgânico em uma só corrente de poder. Quando essa herança foi redescoberta pelas Grandes Casas há gerações, a corrida por seu controle reescreveu a política do continente.

A ferida do presente é o Golpe contra a Casa Varen, há 6 meses. Uma casa de segundo escalão, em ascensão veloz, foi exterminada num único golpe coordenado — porque ameaçava a oligarquia e porque guardava, sem saber, a chave biológica do próprio poder antigo: os Filhos de Varen, imunes à corrupção da fusão.

2. As 3 Eras
🜂 Era Antiga — A Civilização Antes do Nome
Civilização tecnomágica que descobriu Eldara, dominou a Tríade e desapareceu (autoextinção pela fusão excessiva — submerso). Deixaram ruínas com memória digital — as dungeons. Cada dungeon é fragmento da consciência coletiva morta.

🜃 Era Recente — A Era das Sete Casas
Colonização humana, extração de Veridium e Lume Antigo, nascimento da aristocracia: 7 Grandes Casas + ~50 Menores, sob o Concílio das Sete. Formação da Conspiração do Crisol (3 Grandes + ~20 Menores) para derrubar qualquer Casa ameaçadora.

🜄 Era Presente — O Pulso Cortado
Há 6 meses, a Casa Varen caiu em uma noite. Os Filhos foram espalhados pela mãe antes do golpe. A regente Varen foi capturada viva e está sob posse do Líder Supremo do Outro Continente. O Concílio acredita que matou todos os Filhos. O Líder Supremo sabe que escondeu vários — e está caçando.

Voz: sorridente, calculista, sempre com uma piada na manga. Fala rápido, barganha rápido. Nunca conta de onde vem.
Fragmento: traz notícias de fora, objetos raros, e plantado rumor de que algo grande aconteceu com uma casa nobre meses atrás. É o único que conecta a vila ao mundo maior. Pode ou não ser infiltrado do Concílio — a ambiguidade é canônica e nunca totalmente resolvida. Mesmo se ele for leal, o jogador vai suspeitar, e é isso que queremos.
Função mecânica: gate do AH global / economia player-driven (Etapa 4), vende itens raros por ouro, compra excedente da fazenda. Voz cômica/leve que equilibra o tom cozy-sombrio.

Etapa 7+8 — Level Design, Progressão, Quests & Diálogos
Etapa 7 — Níveis & Progressão
Framework: Metroidvania Gating (soft) + Kishōtenketsu por dungeon + Pacing montanha-russa (calm → tense → boss → calm).

7.1 Formato de Mundo: Hub Procedural por Bioma
Modelo: 5 grandes biomas fixos (os arquétipos da Etapa 5), cada um com 25–30 áreas de exploração procedurais entre o coração (vila do jogador) e a fronteira com o bioma vizinho. Não existem portais ou indicadores de "nova zona" — o bioma novo aparece organicamente: a paleta muda, os mobs mudam, as plantas mudam.

[Vila do Jogador] → Área 1 → ... → Área ~12 (miolo do bioma) → ... → Área ~25 (fronteira) → [Bioma Vizinho]
Regras estruturais:

Mapas são seed individuais por jogador — dois jogadores nunca têm o mesmo território procedural
Dungeons existem como pontos fixos no mapa, mas o interior é instanciado por party (PoE-style)
Nyhrr (endgame) é o único território com layout curado e compartilhado — todos os jogadores pisam no mesmo chão lá
Biomas são opcionais para a quest principal — um jogador pode completar a narrativa inteira no bioma natal, mas materiais raros são exclusivos de biomas distantes → o Action House é a ponte econômica para quem não quer explorar
Gating: suave por dificuldade de mobs. O jogador pode tentar cruzar fronteiras no dia 1; o ambiente pune a pressa. A Mochila de Expedição e a Estrela de Sangue amplificam o custo de explorar antes da hora. Isso sustenta o Pilar 2 (Liberdade Modular): o farmer puro compra via AH, o crawler puro vai buscar pessoalmente.

7.2 Biomas & Zonas
Cada bioma tem uma mecânica dominante que diferencia o gameplay de exploração — não só a estética.

🌾 Planície — Vento Seco
Sensação	Trigais ressecados, céu enorme, silêncio absoluto interrompido por galope
Paleta	Amarelo-ocre + azul desbotado. Dia: exposição total. Noite: escuridão plana.
Mecânica dominante	Visibilidade e flanqueamento. Campo aberto = sem cobertura. Inimigos usam formação (bandidos a cavalo cercam, mantis mecanizadas atacam de ângulos). O jogador precisa criar cobertura (obstáculos craftáveis de expedição) ou usar terreno escasso.
Ameaças	Bandidos a cavalo (fast, pursuit), Mantis Mecanizadas (burst de dano, telegrafadas)
Materiais exclusivos	Fibra de Mantis (componente Cyber pra armaduras leves), Grão de Veridium Eólico (componente Orgânico pra buff de velocidade)
Dungeon icônica	A Sementeira Enterrada — ruína da Era Antiga que processava colheitas. Memória digital de abundância. Boss: Colheitadeira Corrompida (AoE de área, força posicionamento constante)
⛰️ Montanha — Pedra Fria
Sensação	Frio cortante, mineração abandonada, eco de cada passo
Paleta	Cinza-azul + laranja de forja. Luz rasante de caverna.
Mecânica dominante	Verticalidade e destruição de terreno. Plataformas de gelo, passagens que desabam, minas com estrutura frágil. Trolls Cibernéticos derrubam paredes — criam atalhos e armadilhas simultaneamente.
Ameaças	Trolls Cibernéticos (tanky, destrói cenário), Avalanches Programadas (evento de área timed)
Materiais exclusivos	Cristal de Lume Frio (componente Mágico pra skills de cryo), Minério de Ferro Cibernético (componente Cyber de maior tier do early-mid)
Dungeon icônica	O Poço Que Respira — mina vertical de 5 andares, cada andar com tema Tríade distinto. Boss: Sentinela de Ferro (lerda, mas bate no mapa — força uso do terreno destruível)
🌿 Floresta — Mato Vivo
Sensação	Luz coada por copa densa, raízes que se mexem quando você não está olhando
Paleta	Verde-escuro + amarelo-fúngico + flash de neon nos druidas corrompidos
Mecânica dominante	Detecção e terreno hostil. As raízes vivas se movem e bloqueiam rotas proceduralmente. Feras Híbridas têm rota de fuga — fogem pra emboscada, não ficam paradas. Druidas corrompidos chamam reforços antes de lutar.
Ameaças	Feras Híbridas (hit-and-run), Druidas Corrompidos (controle de área + suporte), Raízes Vivas (hazard ambiental)
Materiais exclusivos	Seiva de Raiz Viva (componente Orgânico pra regeneração), Osso de Fera Híbrida (componente Cyber+Orgânico pra builds híbridas — raro)
Dungeon icônica	O Coração da Árvore — dungeon que cresce conforme o jogador avança (salas se fecham atrás). Boss: O Druida Primordial (muda de fase somando raízes ao corpo; cada fase destrói uma rota de fuga)
🌊 Costa — Maresia Apodrecida
Sensação	Cais quebrado, gaivotas mortas na areia, naufrágios que deveriam estar parados
Paleta	Azul-cinza + ferrugem + bioluminescência verde nas criaturas do fundo
Mecânica dominante	Maré e profundidade. Certas áreas do mapa ficam acessíveis só na maré baixa (ciclo de 4h real/12h in-game). Naufrágios "ativos" têm IA própria — se movem lentamente, reposicionando salas.
Ameaças	Piratas (organização humana, táticas de grupo), Criaturas do Fundo (surgem de maré alta, imprevisíveis), Naufrágios Ativos (hazard móvel)
Materiais exclusivos	Âncora Rúnica (componente Mágico pesado, pra skills de controle), Escama do Fundo (componente raro pra resistência elementar alta)
Dungeon icônica	O Galeão que Afunda Sempre — dungeon de tempo real: o naufrágio está afundando enquanto você explora. Salas inundam em ordem. Boss: O Capitão Afogado (luta subaquática, limita mobilidade de dodge)
🌫️ Pântano — Bruma Doce
Sensação	Neblina pesada que faz tudo parecer perto, fogo-fátuo que se parece com saída, podridão com cheiro de flores
Paleta	Verde-musgo + roxo-pálido + amarelo de fogo-fátuo. Sem horizonte visível.
Mecânica dominante	Visibilidade reduzida e inimigos que voltam. Neblina limita o alcance de visão do jogador (~40% do normal). Mortos-vivos voltam uma vez se não forem destruídos com dano específico (fogo ou Lume). Necromantes ressuscitam mobs derrotados se não forem priorizados.
Ameaças	Necromantes Menores (suporte tático — ressuscitar é a ameaça real), Bestas dos Brejos (corpo a corpo lento mas pesado), Fogo-Fátuo (hazard de desorientação)
Materiais exclusivos	Fogo-Fátuo Engarrafado (componente Mágico pra dots de fogo), Lama de Bruma (componente Orgânico pra veneno e slow)
Dungeon icônica	A Necrópole Submersa — dungeon onde metade do layout está submerso em lama rasa (movimento lento). Boss: O Arquinecromante (ressuscita todos os mobs que o jogador matou durante a dungeon — o boss fight é um julgamento de quão limpo foi o run)
7.3 Curva de Dificuldade
Pacing geral: montanha-russa. Cada dungeon é um pico de tensão. Cada retorno à vila é o vale de recuperação. A dificuldade sobe a cada novo anel de exploração (distância da vila).

Primeiras 3 horas — "O Chão Familiar" (Áreas 1–8, bioma natal)
Hora	Conteúdo	Tensão
0–30min	Tutorial implícito: vila, primeiro plantio, primeiro NPC (Mara). Sem combate obrigatório.	🟢 Baixa
30–60min	Primeiros mobs nas bordas da fazenda (tier 0 — tutorial de combat). 1ª entrega de quest.	🟡 Crescendo
60–120min	Mapa de exploração 1–4. Primeiro pack de mobs com elite.	🟠 Médio
120–180min	1ª Dungeon narrativa (a porta que abre só pra sangue Varen). Boss tier 1. Revelação de Bren.	🔴 Pico → 🟢 Return (vila)
Design principle nas primeiras 3h: cada mecânica nova aparece primeiro fora da dungeon, num contexto de baixo risco, antes de aparecer dentro. O dodge é apresentado num mob de campo antes do boss. A Mochila de Expedição é apresentada numa missão de coleta antes da 1ª dungeon.

Mid-game — "A Borda do Mapa" (Áreas 9–22, Fazendeiro → Artesão, ~5–25h)
Mobs tier 1 no miolo → tier 2 nas bordas → tier 2+ na fronteira
Introdução de elites com mecânicas únicas (mini-boss de área, não de dungeon)
Primeiras áreas de fronteira de bioma: o jogador vê o bioma vizinho antes de entrar
Dungeons narrativas: A Sementeira Enterrada (mid), O Poço Que Respira (mid-late)
Pico narrativo do mid-game: Revelação 1 (imunidade à Tríade) acontece na 3ª dungeon principal — aqui o crafting tier 2 abre, o jogo muda de velocidade
End-game — "O Que Não Tem Nome" (Áreas 23+, biomas distantes + Nyhrr, Líder → Rei)
Biomas distantes: o jogador entra em território desconhecido com mobs tier 3+
Biblioteca Submersa de Vael'Tor: patamar Líder, gatilho da Revelação 2 (mãe viva)
Nyhrr: território curado, compartilhado, Atlas-style. Cada incursão é um mapa de contenção — o objetivo não é "limpar", é "aguentar e extrair com o máximo possível"
Dificuldade em Nyhrr é assimétrica: quanto mais tempo na incursão, mais o ambiente reage. A fusão Cyber+Carne de Nyhrr aprende o jogador.
7.4 Pontos de Salvamento & Checkpoints
Salvamento: contínuo e automático (MMO-online). Não existe "salvar o jogo" manual.

Checkpoints físicos no mapa de exploração: Postos de Memória — estruturas da Era Antiga (pedras rúnicas) que o jogador ativa ao passar. Uma vez ativado, vira ponto de respawn e fast-travel local (só pra postos já ativados).

Distribuição: 1 Posto a cada 3–4 áreas de exploração
Nas fronteiras de bioma: Posto obrigatório logo antes da travessia
Custo de ativação: 1 fragmento de Lume (material básico) — não é barreira real, é ritual de pertencimento
Dentro de dungeons:

Checkpoints entre andares, nunca entre packs
Morrer num andar retorna ao checkpoint do andar (não ao início da dungeon)
A Estrela de Sangue ainda conta — o checkpoint não cancela a penalidade de morte, só evita que você repita o andar inteiro
A Vila como ancora: sempre acessível via portal de emergência (custo: 1 item da Mochila de Expedição, descartado aleatoriamente). Usar o portal com a mochila cheia é uma decisão agonizante — exatamente o que queremos.

Etapa 8 — Quests & Diálogos
Frameworks: Pirâmide de Freytag (quest principal) + Kishōtenketsu 4 atos (side quests) + Sistema de escolha por Dial de Reputação por Facção.

8.1 Quest Principal — "O Herdeiro da Cinza"
Estrutura pela Pirâmide de Freytag com 5 atos mapeados nos patamares de Reputação:

Ato Freytag	Nome In-game	Patamar	Beat Narrativo
Exposição	"Mãos na Terra"	Errante	Aren chega sem nome declarado. Mara serve sopa sem perguntar. Old Bren olha demais. A vila está quebrada. O jogador planta. A câmera não explica nada.
Ação Crescente	"Brasão Enterrado"	Fazendeiro → Artesão	1ª dungeon abre só pra sangue Varen. Aren toca a porta e ela abre. Bren revela que foi guarda Varen. Lyselle chega com livros proibidos. Revelação 1: imunidade à Tríade destravada. O crafting muda.
Clímax	"Conto de Vael'Tor"	Líder	Biblioteca Submersa: Aren lê o Conto completo. Revelação 2: a mãe está viva, em Nyhrr. O Caçador Sem Nome confirma. O jogador entende que construiu um reino inteiro pra ter exército suficiente pra um resgate.
Queda	"A Porta que Range"	Líder → Rei	A Pedra Mágica abre. Os NPCs da vila ficam pra trás. Aren (e a party, se co-op) entra em Nyhrr. O continente que range não é um lugar — é uma escolha.
Resolução	"Coroa de Cinza"	Rei	Cidadela do Crisol. Confronto com o Líder Supremo (o que Aren poderia ter sido se a fusão o tivesse tocado). Resgate da mãe. A vila vira capital. Aren escolhe o título em cena final — o jogo não nomeia por ele.
Nota de design: a quest principal não tem marcador de "próximo objetivo" permanente. Cada beat narrativo é entregue por um NPC da vila em momento orgânico — você descobre o que fazer porque alguém te conta, não porque um waypoint aparece no mapa.

8.2 Side Quests — Estrutura Kishōtenketsu
Kishōtenketsu: Ki (introdução) → Shō (desenvolvimento) → Ten (twist sem conflito forçado) → Ketsu (conclusão que reconcilia).

SQ1: "A Sopa Que Ela Não Esquece" — Mara, a Taverneira
Ato	Conteúdo
Ki	Mara pede ingredientes raros pra uma receita antiga. Missão de coleta simples. "Um prato que não faço há seis anos."
Shō	Os ingredientes só existem nas bordas do bioma. Enquanto o jogador coleta, Mara conta fragmentos: havia uma mulher na corte que pedia esse prato toda semana, sempre na véspera de uma decisão difícil.
Ten	Ao entregar os ingredientes, Mara para de cozinhar. Olha pro jogador. "Você tem os olhos dela. Sabia desde o primeiro dia." Ela conhecia a mãe de Aren — cantavam juntas antes do Golpe. Fez a sopa pra ele, não pra uma memória.
Ketsu	Jogador pode confirmar ou negar a identidade. Mara aceita qualquer resposta. "Não preciso que você diga. Sei pelos olhos." Recompensa: receita de comida buff exclusiva ("Sopa do Lar" — +15% XP por 2h real) + quests noturnas da taverna desbloqueadas.
SQ2: "O Martelo que Não Esquece" — Old Bren
Ato	Conteúdo
Ki	Bren pede ajuda pra recuperar uma ferramenta perdida numa mina tomada por mobs. Missão de combate direta, parece rotineira.
Shō	A mina tem inscrições rúnicas Varen nas paredes. Bren vai ficando mais quieto a cada andar. Comenta menos. Para de reclamar dos mobs.
Ten	No fundo: um cadáver com a armadura da Guarda Varen — e o martelo. O morto é alguém que Bren conhecia pelo nome. Ele para de falar. 30 segundos de cena muda — só o som da mina. O jogador não pode interagir.
Ketsu	Bren enterra o martelo no chão da praça da vila como memorial. "Ele merecia mais do que uma mina." Destrava crafting tier 2 e, nessa noite, Bren chama o jogador pra forja e revela sua linhagem Varen.
SQ3: "Páginas Que Queimam" — Lyselle Ven
Ato	Conteúdo
Ki	Lyselle perdeu um caderno numa dungeon — fugiu de um agente do Concílio e largou tudo. "Sem aquilo, não tenho prova de nada." Parece missão de recuperação simples.
Shō	Dentro da dungeon: agentes do Concílio vivos, com diálogo. Eles também procuram o caderno. O jogador percebe que não está numa dungeon de mobs — está num teatro político.
Ten	O caderno contém uma lista de nomes: Filhos Varen confirmados, endereços, disfarces. O Concílio sabia de mais do que aparentava. Lyselle estava sendo seguida há meses — e veio pra essa vila porque o nome do jogador está na lista.
Ketsu	Decisão irreversível: queimar a lista (protege os Filhos, mas Lyselle perde a única prova que a mantinha útil pro Concílio — e talvez a viva) ou guardar (prova poderosa, risco imenso pro jogador e pra vila). Consequência narrativa: afeta eventos mid-game com outros Filhos — alguns aparecem mais cedo, outros nunca aparecem dependendo da escolha.
SQ4: "O Rastro do Mudo" — Caçador Sem Nome
Ato	Conteúdo
Ki	O Caçador deixa marcas gravadas em árvores e pedras — setas, símbolos, sinais. Nunca fala. O jogador segue o rastro por instinto (sem quest marker explícito).
Shō	As marcas levam a um acampamento abandonado: cinco fogueiras, cinco colchões, cinco conjuntos de ferramentas. Tudo destruído. Não por mobs — por humanos treinados.
Ten	O Caçador aparece atrás do jogador. Sem falar, puxa a manga: uma marca de linhagem Varen. Mostra os cinco mortos — eram seus irmãos. Rastreou quem os matou até o bioma adjacente. Aponta na direção.
Ketsu	Missão de caça no bioma vizinho — primeiro cruzamento narrativo inter-bioma. O alvo é um Caçador do Líder Supremo. Derrotá-lo revela o primeiro fragmento confirmatório: a mãe está viva. Destrava o co-op narrativo (o Caçador Sem Nome vira party-member opcional).
SQ5: "Ouro com Cheiro de Queimado" — O Mercador Itinerante
Ato	Conteúdo
Ki	O Mercador quer ajuda pra recuperar uma "carga perdida" em área de exploração perigosa. Pagamento generoso. Não diz o que é a carga.
Shō	A carga são caixas lacradas. Ao recuperá-las, algumas caixas estão abertas — os documentos dentro são registros de suborno: nomes de Casas Menores, valores, datas, selos do Concílio.
Ten	O nome do Mercador aparece nos documentos. Como informante ativo. Ele estava na vila antes do jogador chegar. A ambiguidade não é acidente — é a função dele.
Ketsu	O Mercador não nega, não confirma. "Informante não escolhe lado. Escolhe preço. Seu preço, até agora, foi conveniente." Escolha: confrontar (perde acesso ao AH desse Mercador por 7 dias reais, mas ganha +10 Rep. Resistência) ou ignorar (mantém relação, mas o jogador sabe — e o Mercador sabe que o jogador sabe). A ambiguidade é canônica e permanente — nunca resolve.
8.3 Árvore de Diálogos — Conversa-Chave
Cena: Old Bren, após a 1ª dungeon, entrega o fragmento do brasão Varen ao jogador. É noite. A forja está acesa. Ele não fala primeiro — espera o jogador chegar perto.

[Bren] (segurando o brasão, de costas): "Esse símbolo... não deveria existir mais. Faz seis meses que eu não via esse ferro."

(pausa)

[Bren]: "Onde você achou isso?"

Escolha A — "Numa dungeon. Tava num baú."
Aren finge ignorância total.

[Bren] (longa pausa, olha pro brasão, depois pro jogador): "Pega. Guarda. Não mostra pra ninguém."

→ Bren fecha o assunto. Mas planta um evento futuro: em 3 dias in-game, ele manda mensagem chamando o jogador pra forja de madrugada. Só aí conta.

Consequência: acesso ao crafting tier 2 com delay narrativo. A conversa real acontece depois, com mais contexto acumulado — Bren conta mais porque o jogador "provou" que sabe guardar segredo ao fingir não saber nada.

Escolha B — "Onde você viu isso antes?"
Aren demonstra curiosidade sem se comprometer.

[Bren] (senta no banco da forja): "Forjei armas pra Guarda Varen por dezesseis anos. A última encomenda foi entregue dois dias antes do Golpe. Nunca recebi o pagamento. Nunca fui atrás."

(olha pras próprias mãos)

"Esse brasão significa que alguém sobreviveu. E que foi parar numa dungeon em vez de num palácio. O que me diz que a história não acabou como pensavam."

→ Bren conta a história da Guarda Varen, a noite do Golpe, o que viu queimar.

Consequência: destrava fragmento de lore "A Noite do Golpe" + Bren vira quest-giver primário (acesso ao crafting tier 2 sem requisito adicional de Reputação). Caminho mais informativo, mas Bren não vai além do que sabe.

Escolha C — "Você sabe quem eu sou."
(disponível só se Rep Fazendeiro ≥ 100 ou SQ2 concluída)

Aren para de fingir.

[Bren] (para de se mover. Fecha a forja. Acende uma segunda vela): "Sabia desde o primeiro dia. Você tem o jeito de plantar dela — varre a terra antes de enfiar a semente. Aprendi isso observando a mãe de você na fazenda de ensino da Casa."

(virar pro jogador)

"Esperava você chegar sozinho nisso. Mas tá bem. Já esperou o suficiente."

→ Bren revela tudo de uma vez: a linhagem, o que sabe do Golpe, o próprio vínculo com os Varen.

Consequência: pula o delay narrativo da Escolha A e o limite de informação da Escolha B. Destrava crafting tier 2 imediatamente + ativa o beat "Encontro com o Mentor" da Jornada do Herói (Etapa 6). Caminho para jogadores que já reuniram contexto suficiente — a Escolha C recompensa curiosidade anterior.

8.4 Sistema de Escolha
Modelo: Dial de Reputação por Facção (3 eixos) + Memória Local de NPC
Três medidores independentes (não somam zero — cada um tem lógica própria):

Facção	Medidor	Como sobe	Como cai
🜚 Concílio do Crisol	0–100	Quests ambíguas de mercador, deliveries sem confirmar destinatário, ignorar dungeons narrativas	Cada dungeon do Crisol derrubada, cada Filho resgatado, revelar identidade pra NPC
⚙️ Líder Supremo	0–100	Contratos de incursão em Nyhrr aceitos, vender informação de Filhos localizados	Resgatar Filhos presos, destruir postos de Nyhrr em Eldara
🌿 Resistência Varen	0–100	Quests dos NPCs da vila, revelar identidade, resgatar Filhos, completar as 5 SQs	Vender acesso, colaborar com Concílio por benefício próprio
Por que não binário: Aren é um Varen — a Rep. com Concílio e Líder Supremo nunca sobe o suficiente pra "virar de lado". Mas pode subir o suficiente pra abrir rotas alternativas: diálogos únicos com agentes infiltrados, atalhos em dungeons do Concílio, informações que só quem está "quase dentro" acessa. O dial não é sobre trair — é sobre quanto você usa o inimigo antes que ele te use de volta.

Memória Local de NPC (sistema complementar):

Cada NPC da vila tem memória de 3 eventos que o envolveram diretamente
Esses 3 eventos determinam o estado relacional: confiante / neutro / desconfiante
O estado muda o tom das conversas, os diálogos disponíveis, e quais quests o NPC oferece
Não é um número visível — o jogador sente, não mede. Bren desconfiante fala menos e olha pro lado. Mara confiante serve comida sem pedir moeda.
Consequências irreversíveis (camada adicional):

Algumas decisões (ex: queimar a lista da SQ3) são permanentes e sem undo
O jogo nunca pede confirmação dupla — a escolha acontece no mesmo ritmo da conversa
Filosofia: o peso da escolha vem da irreversibilidade, não da pontuação moral. Não existe escolha "boa" ou "má" marcada — existe escolha feita, e o mundo que segue daí.

Lockout de Nyhrr vs lockout de dungeons normais — a Estrela de Sangue tem regra diferente pra morte de boss (mencionada na Etapa 4 como "regra separada"). Precisa de especificação antes de design detalhado de Nyhrr.
Co-op e quests — as SQs aqui são desenhadas pra experiência solo. Como a memória de NPC e as escolhas irreversíveis funcionam em party de 2–6 jogadores? Decisão de design ainda aberta.
Outros Filhos como NPCs — o canon da Etapa 6 indica que outros jogadores são canonicamente outros Filhos. A SQ4 tangencia isso. A estrutura de quests inter-player ainda não está mapeada.

Etapa 8 — Quests & Diálogos
Etapa 8 — Quests & Diálogos
Frameworks aplicados: Pirâmide de Freytag (quest principal) · Kishōtenketsu (side-quests) · Sistema Witcher A/B (escolhas sem âncora moral). NPC-givers canônicos: Aldric (líder da vila) · Bren (ferreiro) · Ducard (taverneiro) · Mara (filha do taverneiro) · Selwyn (mercador itinerante). Regra universal de escolha: o jogador nunca sabe qual foi a decisão "certa" no momento em que escolhe. O feedback chega tarde — e às vezes nunca.

8.1 Quest Principal — "O Nome que Eu Escondi"
Estrutura: Pirâmide de Freytag em 5 fases, distribuída ao longo de toda a progressão. Tema: vingança não é destino — é consequência de ter construído algo que vale proteger.

Fase 1 — Exposição (Errante)
Aren chega à vila sem nome verdadeiro, com as mãos no bolso e o olho no chão. Aldric, o líder pragmático, dá a terra sem cerimônia: "Aqui não perguntamos de onde você vem. Mas esperamos que você plante." Mara aparece no dia seguinte com sementes que ninguém pediu que ela trouxesse. Bren observa da porta da forja sem falar.

A vila está decadente: dois prédios funcionando, quatro com telhado de buraco. O jogador planta, entrega, constrói — e a vila nasce.

Gatilho narrativo: na primeira noite, Ducard dá um quarto sem cobrar. Sem explicação.

Fase 2 — Ação Crescente (Fazendeiro → Artesão)
A vila cresce via quests. Novos NPCs chegam. Bren pede que Aren entre numa dungeon cujas portas não abrem pra ninguém além dele.

Dentro da dungeon (A Sementeira Enterrada), Aren toca a Tríade — Cyber, Magia e Orgânico ao mesmo tempo — e não é corrompido. Bren, do lado de fora, reconhece o que isso significa: não é sorte, é sangue.

Bren revela ser um Varen de geração anterior. Conta fragmentos. Não conta tudo.

Casas Menores começam a aparecer nas bordas — emissários com ofertas vagas de aliança. Algumas são genuínas. Outras são olhos do Concílio. O jogador não tem como saber qual é qual ainda.

Destrava mecânico: crafting tier 2 + armas exclusivas (Tríade).

Fase 3 — Clímax (Líder)
A Biblioteca Submersa de Vael'Tor abre. Dentro, o Conto dos Filhos de Varen em forma original: a mãe não morreu. Está viva. Em Nyhrr. Sob posse do Líder Supremo.

Imediatamente depois: uma das 3 Grandes Casas do Crisol confirma que Aren existe e manda agentes à vila.

Escolha Witcher — sem âncora moral:

A) Confrontar os agentes abertamente → a vila sofre dano estrutural (2 prédios regridem, Aldric fica ferido, quests dele ficam indisponíveis por período real), mas o Concílio recua temporariamente e a identidade de Aren se torna conhecida entre as Casas Menores — algumas se aproximam, outras somem.

B) Recuar e desaparecer por um tempo → a vila fica intacta, mas o Concílio marca a região como "verificada" (novos agentes voltam mais cedo no endgame). Os aliados de Casas Menores que não recebem resposta de Aren assumem abandono — 1-2 alianças se dissolvem silenciosamente.

Não há opção C. Não há opção "certa".

Fase 4 — Queda (Líder → Rei)
A pedra mágica está destravada. Nyhrr espera. Aren entra num território onde a fusão aconteceu sem freio — cidades que respiram, soldados meio aço, meio pulmão.

Aqui a quest principal converge com o endgame Atlas-style: cada incursão em Nyhrr é um passo em direção à Cidadela do Crisol.

Fase 5 — Resolução (Rei)
Cidadela do Crisol. O Líder Supremo está fundido ao seu trono-máquina. Sofrendo.

A mãe está viva. Refém. À vista.

Escolha final:

A) Matar o Líder Supremo → vingança direta. A fusão colapsa com ele — a Cidadela desmorona, a saída é por tempo. A mãe é resgatada. Aren chega à vila-capital como herdeiro. A coroa é restaurada. O Concílio sobrevive sem sua cabeça mais radical — o que acontece a seguir, o jogador nunca vê.

B) Oferecer ao Líder Supremo a morte como libertação → ele pede isso há anos. A fusão está consumindo a consciência dele. Aren o liberta de propósito, não de ódio. A Cidadela ainda desmorona. A diferença é interna — e o jogo não a celebra. A mãe é resgatada. A coroa, restaurada.

Nenhuma opção é recompensada com mais recursos. A diferença é o que Aren carrega pra casa.

8.2 Side-Quests
Estrutura Kishōtenketsu em 4 atos: Ki (introdução) · Shō (desenvolvimento) · Ten (twist) · Ketsu (conclusão com escolha A/B). NPC-givers são os 5 NPCs da vila.

SQ1 — "A Terra que Aldric Nunca Dividiu"
NPC-giver: Aldric (Líder da Vila)

Ato	Conteúdo
Ki	Aldric pede ajuda pra resolver disputa entre dois moradores sobre terra na borda da fazenda. Rotineiro.
Shō	Investigando, Aren encontra uma ruína enterrada naquela terra — câmara mortuária com símbolo da Casa Varen.
Ten	Aldric sabe. Sempre soube. Trinta anos atrás, um Varen fugindo pediu refúgio. Aldric deu. O Varen morreu de ferimentos e foi enterrado ali. Aldric nunca contou a ninguém — não por culpa, mas por peso.
Ketsu — A	Pedir que Aldric revele à vila. Ele revela. Ganha +15 Rep · Aldric fica abalado — quests dele ficam indisponíveis por 5 dias reais. A vila reage com silêncio, depois com respeito.
Ketsu — B	Guardar segredo com Aldric. Relação se aprofunda. Aldric abre acesso a uma área da vila que mantinha trancada — guarita da Era Antiga, com material exclusivo.
SQ2 — "O Martelo e a Marca"
NPC-giver: Bren (Ferreiro)

Ato	Conteúdo
Ki	Bren pede que Aren entregue uma faca especial a um ferreiro numa vila ao norte. "Só entrega. Não abre."
Shō	O ferreiro do norte está morto. A casa foi revistada. A faca tem uma runa embaixo do cabo — símbolo de comunicação entre descendentes Varen. Alguém chegou antes.
Ten	Num forno apagado, escondida, uma lista de 7 nomes. O nome de Aren está lá — com a vila correta. Alguém já sabia onde ele estava antes de Bren precisar entregar qualquer faca.
Ketsu — A	Compartilhar a lista com Bren abertamente. Ele começa a fazer contato com os outros Filhos — abre a rota narrativa para o co-op canônico (outros jogadores como irmãos encontrados). Informações chegam mais devagar, mas a rede cresce.
Ketsu — B	Guardar a lista e investigar sozinho. Aren avança mais rápido nas revelações, mas Bren fica fora do circuito. A rede de Filhos não se forma — co-op narrativo fica limitado a encontros casuais, sem o peso emocional da irmandade.
SQ3 — "Mara Quer Ver o Mar"
NPC-giver: Mara (Filha do Taverneiro)

Ato	Conteúdo
Ki	Mara quer que Aren a acompanhe até a borda da floresta pra ver o pôr do sol "de cima". Parece passeio.
Shō	No caminho: acampamento abandonado de espiões do Concílio. Relatórios sobre a vila. O nome "Aren" com uma interrogação ao lado.
Ten	Mara já sabia. Ela rastreou os espiões sozinha durante semanas. Não precisava de escolta — precisava de testemunha. Ela entrega os relatórios a Aren: "Você tem um prazo. Eles não têm certeza ainda. Mas vão ter."
Ketsu — A	Contar pra Mara quem você é. Ela não foge. Fica. Torna-se aliada ativa — quests mais profundas desbloqueadas, ela começa a monitorar agentes do Concílio na região. O risco emocional aumenta: agora tem alguém na vila que morre se o Concílio confirmar a identidade de Aren.
Ketsu — B	Agradecer sem revelar. Mara respeita o silêncio — mas a relação congela num patamar de "amizade com distância". Ela continua monitorando por conta própria. Sem profundidade narrativa adicional.
SQ4 — "A Casa que Não Tem Nome"
NPC-giver: Emissário de Casa Menor Anônima

Ato	Conteúdo
Ki	Um emissário chega à taverna com proposta: apoio logístico (materiais, inteligência sobre o Concílio) em troca de "lealdade futura não especificada". Sem detalhes, sem nome da Casa.
Shō	Investigando: a Casa Menor foi quase destruída pelo Concílio há 10 anos. Sobreviveu virando informante. O apoio oferecido é real — e a Casa também reporta posições de aliados ao Concílio. Duplo agente por sobrevivência, não por malícia.
Ten	O emissário sabe que Aren descobriu. Diz, sem desviar o olhar: "Toda Casa que sobreviveu ao Crisol tem essa mancha. A questão é: qual mancha você consegue viver com?"
Ketsu — A	Aceitar o acordo. Materiais e inteligência chegam regularmente. O Concílio recebe relatórios vagos sobre a localização de Aren — curto prazo conveniente, longo prazo perigoso (os agentes do clímax da quest principal chegam mais rápido).
Ketsu — B	Recusar e propor terceira via: a Casa para de informar ao Concílio em troca de proteção futura. Sem benefícios imediatos — mas cria aliada genuína. O Concílio recebe um "silêncio suspeito" nessa região, o que também chama atenção. Não existe saída limpa.
SQ5 — "Ouro com Cheiro de Queimado"
NPC-giver: Selwyn (Mercador Itinerante)

Ato	Conteúdo
Ki	Selwyn quer ajuda pra recuperar uma "carga perdida" em área de exploração perigosa. Pagamento generoso. Não diz o que é.
Shō	A carga são caixas lacradas. Algumas estão abertas — os documentos dentro são registros de suborno: nomes de Casas Menores, valores, datas, selos do Concílio.
Ten	O nome de Selwyn aparece nos documentos. Como informante ativo. Ele estava na vila antes de Aren chegar. A ambiguidade não é acidente — é a função dele.
Ketsu — A	Confrontar Selwyn. Ele não nega, não confirma. "Informante não escolhe lado. Escolhe preço. Seu preço, até agora, foi conveniente." Aren perde acesso ao AH desse Mercador por 7 dias reais. Ganha +10 Rep com a facção de Resistência (Casas Menores leais). A ambiguidade permanece canônica — o jogo não resolve.
Ketsu — B	Ignorar. Manter a relação. Selwyn sabe que o jogador sabe — e o jogador sabe que Selwyn sabe que sabe. O AH continua aberto. A tensão existe permanentemente sob toda transação futura. Também nunca resolve.
8.3 Árvore de Diálogos — Conversa-Chave com Bren
Cena: Noite. Forja. Bren segura o brasão Varen. Aren acabou de não ser corrompido pela Tríade na 1ª dungeon. Tom: quieto, pesado, sem trilha sonora no início — só o som da brasa.

BREN: "Você tocou o Lume Antigo, o circuito e a carne ao mesmo tempo. E está de pé. Sabe quantas pessoas consigo contar nos dedos que fizeram isso?"

Escolha A — "Eu não sabia o que estava fazendo."
Tom do jogador: vulnerável, honesto.

BREN (algo que parece alívio): "Melhor ainda. Quem toca sem saber é mais Varen do que quem tenta. A Tríade não obedece intenção — obedece linhagem."

Ele coloca o brasão na mão de Aren.

"Isso é seu. Não porque eu decidi — porque o metal reconheceu."

Consequências:

Bren explica a imunidade Varen imediatamente e com detalhes.
Crafting tier 2 e armas exclusivas são desbloqueados com 1 dungeon de antecedência em relação ao ritmo normal.
Bren passa a ser mais aberto em quests futuras — responde perguntas que normalmente desviaria.
Relação com Bren: +10.
Escolha B — "Quantas?"
Tom do jogador: calculista, foca na pergunta.

Bren para. Olha a forja. A brasa crepita.

BREN: "Sua mãe. Você. E mais uns poucos que espero que ainda estejam plantando cenouras em algum lugar longe daqui."

Ele não entrega o brasão. Guarda no avental.

"Por hoje é isso."

Consequências:

A revelação da mãe viva é antecipada — sem localização, sem confirmação, mas a semente está plantada.
A quest de mid-game "A Mãe Viva" é ativada antes do patamar Líder (ritmo normal), com menos informação — o jogador chega ao patamar Líder já procurando ativamente.
Crafting tier 2 desbloqueado no ritmo normal.
Relação com Bren: +5 (ele testou, Aren passou, mas o muro ainda está lá).
Escolha C — "Para. Você me conhece. Me diga quem você é primeiro."
Tom do jogador: assertivo, coloca o mentor em xeque.

Bren larga o martelo. Pausa longa — a mais longa do jogo até aqui.

BREN: "Sou o que sobrou de um Varen que escolheu o martelo em vez do nome. Quarenta anos. Sua mãe sabia onde eu estava. Por isso você está aqui e não num fosso."

Ele não entrega o brasão. Senta. Parece mais velho de repente.

"Agora me diz: você veio pra plantar ou veio pra morrer?"

→ Aren não responde. O diálogo fecha. Bren volta ao trabalho.

Consequências:

Bren revela o arco completo do mentor — guardião designado pela mãe, 40 anos esperando.
SQ2 ("O Martelo e a Marca") é ativada imediatamente, sem esperar o patamar Fazendeiro.
Bren torna-se companion ocasional em dungeons da Floresta (NPC de suporte que aparece no mapa, opcional de usar).
A explicação sobre a Tríade não vem nessa cena — Bren não volta ao assunto até Aren iniciar a SQ2.
Relação com Bren: +20 (o mais alto possível nessa cena — mas é um vínculo com peso, não com calor).
8.4 Sistema de Escolha
Parâmetro	Decisão
Modelo	Witcher A/B — binário, sem opção neutra
Feedback	Consequências chegam tarde — horas de jogo depois, às vezes só no clímax
Sinalização moral	Nenhuma. Sem ícone de "bom/mau", sem cor de opção, sem música de reforço
Travamento de progressão	Nunca. Toda rota chega ao mesmo clímax — com aliados e ferramentas diferentes
Memória do sistema	O jogo rastreia escolhas silenciosamente. Emissários de Casas Menores aparecem ou somem nas tavernas sem explicação — o ambiente sinaliza, o UI não
Reversibilidade	Impossível. Escolhas são permanentes dentro de cada playthrough
Princípio de design
"O jogo não julga. Os NPCs, às vezes, sim — mas só muito depois."

Nenhuma escolha é recompensada com mais recursos do que a outra por design deliberado. A diferença está em quem está ao seu lado quando a vila é atacada e o que Aren carrega quando entra na Cidadela.

8.5 Nota de Reconciliação — Reputação × Crescimento da Vila
Cânone vigente (Etapa 4): Reputação é o gate de progressão universal, ganha por combate, farming e crafting. Decisão desta etapa: a vila cresce somente via entrega de quests.

Reconciliação adotada:

Reputação (combate + farming + crafting)
    → pré-requisito para desbloquear quest de nível Y

Quest entregue → vila cresce (prédio novo, NPC novo)
Um jogador pode acumular Rep alta fazendo só dungeons — mas sem entregar quests, a vila não avança. Rep alta abre o catálogo de quests disponíveis; entregar a quest é o ato construtivo. Isso honra o Pilar 1 (Reconstrução) e os Pilares 2 e 3 (Liberdade Modular e Risco com Peso): cada playstyle gera Rep, mas a vila pede que o jogador apareça — não apenas que jogue.

Etapa 9 — Direção de Arte
Etapa 9 — Direção de Arte
Framework aplicado: Moodboard Iconografia · Silhouette Test · Teoria de Cores de Itten (contraste simultâneo + complementares) Resolução canônica de sprite: 64×64 px Estilo: Pixel art isométrico

3 Palavras de Mood
ÉPICO · BARROCO · MELANCÓLICO

Épico — escala política da narrativa, grandiosidade tática dos campos e da trilha; o visual precisa sustentar a sensação de que algo grande está em jogo.
Barroco — ornamento como identidade; vestimentas com camadas, arquitetura com molduras, complexidade que remete à arte sacra e monárquica europeia (DNA direto de Akihiko Yoshida).
Melancólico — paleta terrosa e desbotada, expressão sem nariz/pupila (Yoshida), peso de perda histórica. O jogo é bonito como uma catedral em ruínas.
Resolução Dual-Tone (Canon da Etapa 1 reconciliado)
Decisão canônica — Opção A: "O Barroco É a Ruína"

O split visual não é hard cut entre dois mundos — é contaminação progressiva:

Zona	Visual
Vila / Fazenda	Arquitetura barroca quente e intacta. Iluminação dourada Rembrandt. Cor de ocre e creme. NPCs com silhuetas arredondadas.
Dungeon early	Mesma arquitetura barroca — mas com veias de circuito crescendo como trepadeiras. Colunas com sulcos luminosos. Vitrais que pulsam dado. Tecnologia jovem, ainda contida.
Dungeon mid/end	A corrupção domina. Colunas metade pedra, metade tubo. O ciano e o âmbar envelheceram — não é neon saturado, é bronze que apodreceu e ficou eletrizado. A Era Antiga ainda pulsa, mas faz isso como uma ferida velha, não como uma tela nova.
Nyhrr (endgame)	Inversão total: tecnologia é a estrutura, o orgânico é o parasita. Carne crescendo sobre metal. A paleta fria domina com acentos de vinho Varen.
Regra de ouro: quanto mais longe da vila, mais a pedra cede ao circuito. O jogador sente a corrupção da Tríade no próprio visual da tela.

Tier 0 (tutorial): Triângulos pequenos e previsíveis — bandidos com ângulos simples, sem ornamento.
Tier 1–2 (mid biomas): Triângulo com elemento circular no centro — ameaça com "coração" exposto (ponto fraco visível na silhueta).
Bosses: Círculo central rodeado de triângulos — o core pulsante (vulnerabilidade) cercado de espinhos. Linguagem visual de "chegue perto, mas com cuidado."
Dungeon mid/end (corrompidos): Shapes barrocas (arcos, colunas) perfuradas por triângulos de circuito. A arquitetura vira inimigo.
Facções (leitura rápida de afiliação):

Concílio do Crisol: triângulo triplo sobreposto (três Casas) em dourado frio
Casa Varen: círculo com losango interno em Vinho Varen — pertencimento + poder contido
Nyhrr / Corrompidos: nenhum shape puro — sempre mistura de dois shapes incompatíveis, simbolizando a fusão incontrolada
Regras de Produção
Vila nunca usa Cobalto Corrompido. Mesmo à noite, a paleta fria da vila usa azul-índigo quente, não o cobalto de dungeon.
Dungeons nunca usam Ocre Varen puro. Se houver ocre nas dungeons, é o Âmbar Eletrizado — a versão corrompida da mesma família de cor.
Vinho Varen aparece uma vez por cena — se aparecer duas vezes, perde impacto narrativo.
Todo novo personagem passa no Silhouette Test antes de colorizar: converter o sprite para 2 cores (preto/branco) e validar leitura de identidade.
Circuitos nas dungeons crescem em direção horizontal/diagonal — nunca verticais puras. Vertical é shape de construção intencional; horizontal/diagonal é shape de invasão orgânica.

5. Lógica de Mixagem
Vertical Remixing — Stems por Estado
Princípio: o jogador nunca ouve uma trilha "trocar". As camadas entram e saem imperceptivelmente conforme o estado muda.

Vila / Fazenda
Estado	Stems ativos
Início do jogo (Errante, 0 prédios)	Stem 1: Cello solo — o tema da vila incompleto
+1 prédio construído	Stem 2: Cordas de câmara entram em harmonia
+3 prédios (patamar Fazendeiro)	Stem 3: Flauta/alaúde adiciona melodia secundária
+5 prédios (patamar Artesão)	Stem 4: Brass discreto na harmonia de fundo
Vila avançada (Líder+)	Stem 5: Percussão suave de marcha — a vila tem ritmo próprio agora
Gatilho: cada prédio construído (SFX 1 → stem ativado permanentemente).

Exploração de Mapa
Estado	Stems ativos
Safe (sem mob no raio de detecção)	Stem 1: Cello drone + textura ambiental de bioma
Tensão (mob detectado, não em combate)	Stem 2: Harmônicos dissonantes de cordas entram
Perigo iminente (elite ou emboscada)	Stem 3: Pulso do Crisol sutil no grave
Dungeon — Combate
Estado	Stems ativos
Pack normal	Stem 1: Cello pizzicato + percussão seca
Elite ou múltiplos packs	Stem 2: Brass marcial entra
Boss fight	Stem 3: Full orchestra + coro litúrgico fragmentado (tema do Concílio sangra)
Última fase de boss (HP crítico do boss)	Stem 4: Tema da Casa Varen fragmentado entra no registro grave — o que está em jogo
Horizontal Re-sequencing — Transições
Transição	Lógica
Vila → Dungeon	Crossfade lento de 8s. O cello mantém a nota — mas o processamento cresce gradualmente (reverb de câmara, bow pressure). O calor sai, o frio entra. SFX 2 (Pulso do Crisol) na metade do crossfade.
Dungeon → Boss	Corte seco de silêncio de 1s → entrada do tema do Concílio/antagonista em volume pleno. O corte é a surpresa. O silêncio é o medo.
Boss derrotado → Saída	Música para. Silêncio de 2s. Cello solo retorna com o tema do protagonista — só a nota que não resolve. O peso continua mesmo na vitória.
Morte (Estrela de Sangue)	Música corta instantaneamente. SFX 7 (sino + vinyl). Silêncio de 3s. Tema do protagonista retorna em tom uma terça mais baixa — a ferida cresceu.
Revelação da Tríade	Música para. SFX 6 (Ativação da Tríade, 2s). Tema da Casa Varen completo pela primeira vez — a resolução que o tema do protagonista nunca havia tido.
Vila → Mapa de Exploração	Crossfade suave de 5s. O Stem da Vila vai baixando enquanto o drone de exploração do bioma vai subindo. Sem corte perceptível.
Nyhrr (entrada)	Versão distorcida do tema da vila começa a sangrar por baixo da música atual antes da transição. O jogador reconhece antes de perceber. Transição de 12s.

13.4 — Como Continuar: Rota Solo + Claude Code
Para você: solo dev com 4–6h/dia, Claude Code como pair programmer, agentes de IA especializados para arte e áudio.

Princípios da Rota Solo com IA
1. Claude Code é seu CTO Use-o como revisor de arquitetura a cada sprint. Antes de implementar qualquer sistema novo, descreva o design para o Claude e peça conflitos com o que já existe. Ele sabe o GDD completo — vai barrar decisões que contradizem o cânone.

2. Pixellab é seu Art Team Todos os 13 prompts da Etapa 12 estão prontos para cópia. Workflow recomendado: gerar variações em batch (5–8 por prompt), silhouette test manual nas top-3, escolher 1, importar. Tempo médio por asset: 2–4h incluindo iterações.

3. ElevenLabs é seu Composer Temas e SFX prontos na Etapa 11. Workflow: gerar 3 variações por prompt, ouvir com o contexto do jogo em mente (não como música isolada), escolher 1, implementar no Godot com vertical remixing (stems por estado).

4. Godot 4 como engine Razões canônicas: isométrico 2D nativo, GDScript próximo de Python (fácil para Claude Code assistir), F/OSS sem royalties, comunidade de pixel art isométrico ativa. Não troque de engine no meio do projeto.

Workflow de Sprint Recomendado (2 semanas)
DIA 1–2   → PLANEJAMENTO
  • Revisar backlog com Claude Code
  • Definir 1 entregável principal + 2 secundários
  • Identificar dependências de asset antes de codar

DIA 3–8   → EXECUÇÃO
  • Manhã (2–3h): código/implementação no Godot
  • Tarde (1–2h): geração/iteração de assets (Pixellab/ElevenLabs)
  • Nunca misturar: código em bloco, assets em bloco

DIA 9–12  → INTEGRAÇÃO
  • Assets gerados entram no Godot
  • Teste funcional do entregável principal
  • Bug fixing guiado pelo Claude Code

DIA 13–14 → REVISÃO
  • Playtest pessoal com critério emocional (não técnico)
  • Registrar o que "não sente certo" — é mais valioso que registrar bugs
  • Decidir o que o próximo sprint resolve

Passo	Ação	Tempo estimado

Quando Reconsiderar a Rota Solo
Os seguintes sinais indicam que a rota solo precisa ser reavaliada — não necessariamente abandonada, mas revisada:

Fase 2 atrasa mais de 4 semanas do planejado sem causa externa clara
A implementação do co-op na Fase 3 exige mais de 6 semanas sem progresso visível
O economy balancing do AH global consome mais de 20% do tempo de desenvolvimento total
Burnout identificado: dois sprints consecutivos sem entregável validado
Nesses casos, a alternativa não é "contratar um time completo" — é contratar 1 dev Godot freelancer por 4–8 semanas para destravar o bloqueio específico. Budget-first: uma consultoria de 40h com dev Godot especialista vale mais do que 3 meses de luta solo num sistema específico.

O Vertical Slice está aprovado e o projeto avança para a Fase 3 quando os 3 critérios abaixo forem satisfeitos simultaneamente:

Critério de Feeling: 3 pessoas que nunca viram o jogo conseguem completar o ciclo fazenda → dungeon → boss → vila sem instrução verbal, e reportam que "entenderam o que o jogo é" ao final.

Critério de Emoção: pelo menos 2 das 3 pessoas reportam alguma versão de "não queria parar de jogar" — independente de bugs ou polish incompleto.

Critério Técnico: o ciclo roda por 30 minutos consecutivos sem crash, com 60fps estável em hardware de desenvolvimento.

Se os critérios 1 e 3 passam mas o critério 2 falha, não avance. O feeling é o produto. Um Vertical Slice tecnicamente correto que não emociona é um diagnóstico de design, não um bug.
